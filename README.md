# Arcade

## Dependencies

- https://miniaud.io/index.html
- sdl2
- sdl2_image
- SFML
- ncurses

## Game implemented

- Snake
- Nibleur

## Graphical lib wrapped

- SDL2
- SFML
- ncurses

## Objectif

Change game/graphical lib at runtime

## Install

```bash
cmake .
make
```

## Usage

```
./arcade ./lib/arcade_sfml.so
./arcade ./lib/arcade_sdl2.soo
./arcade ./lib/arcade_ncurses.so
./arcade
```
